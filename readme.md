# 🚀 VM-SEHGORP HTML Code Editor

A simple and responsive **HTML live code editor** that runs directly in your browser. Built with pure HTML, CSS, and JavaScript — no frameworks or external libraries required.

---

## 🎯 Features

✅ Real-time HTML preview  
✅ Download your code as `.html` with one click  
✅ Mobile-friendly responsive design  
✅ Clean and distraction-free interface  
✅ Lightweight, no installation needed

## 📋 How to Use

1. **Open `index.html`** in any modern browser (Chrome, Edge, Firefox, etc.)
2. Start typing your HTML code in the editor.
3. Preview is updated in real-time in the bottom pane.
4. Click the **Download** button to save your code as `index.html`.
